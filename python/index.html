🐍 Python (Desktop & Data Analyst)

Bahasa yang mudah dipelajari dan sangat fleksibel untuk banyak kebutuhan.

Fullstack, Mobile & Desktop

📖 Penjelasan Mendalam

Python populer karena sintaksnya ringkas dan mudah dibaca. Materi mencakup dasar pemrograman, struktur data, fungsi, serta pengantar analisis data menggunakan pustaka seperti pandas.

💡 Analogi & Contoh

Python seperti alat serbaguna: mudah dipakai untuk tugas harian hingga proyek data yang kompleks.

💻 Contoh Praktik

nilai = [80, 90, 75]
rata = sum(nilai) / len(nilai)
print(f'Rata-rata: {rata}')

✏️ Latihan Soal

  1. Buat program input 5 nilai lalu hitung rata-ratanya.
  2. Buat script menampilkan data siswa dalam format tabel teks.
  3. Coba baca file CSV sederhana lalu tampilkan 3 baris pertama.
