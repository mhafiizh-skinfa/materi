🛠️ Section 3: Contoh step-by-step

Langkah 1: Variabel

Menampilkan data sederhana.

# Menampilkan identitas
nama = "Salsa"
print(f"Halo {nama}")

Langkah 2: Loop

Iterasi data produk.

produk = ["Roti", "Susu", "Kopi"]
for item in produk:
    print(item)

Langkah 3: Function

Membuat fungsi hitung diskon.

def hitung_total(harga, diskon):
    # Menghitung total setelah diskon
    return harga - (harga * diskon / 100)
