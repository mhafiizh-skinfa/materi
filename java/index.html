Kelas XI

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Konsep OOP: class, object, method, encapsulation.
  • Tipe data statis dan kontrol alur dasar.
  • Constructor dan overloading sederhana.
  • Penerapan Java untuk aplikasi bisnis/enterprise.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
ClassCetak biru objek.Gunakan PascalCase.
ObjectInstansi class.Isi properti melalui constructor.
MethodPerilaku objek.Satu method satu tujuan.

Java OOP wajib

Dasar OOP: class, object, constructor, method. Empat pilar OOP dengan analogi: encapsulation (kotak hitam), inheritance (warisan sifat), polymorphism (satu perintah banyak bentuk), abstraction (hanya fitur penting).

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • Class seperti blueprint kendaraan, object kendaraan jadinya.
  • Method seperti tombol fungsi pada mesin cuci.

Kasus Dunia Kerja

Sistem core perbankan banyak dibangun dengan Java karena stabil dan terstruktur.

Koleksi dan error handling

Gunakan ArrayList dan HashMap untuk data dinamis, serta try-catch-finally untuk penanganan error aman. Sertakan file I/O baca/tulis file.

🛠️ Section 3: Contoh step-by-step

Langkah 1: Program awal

Cetak teks ke konsol.

public class Main {
  public static void main(String[] args) {
    // Menampilkan salam
    System.out.println("Halo Java");
  }
}

Langkah 2: Class object

Buat class sederhana.

class Siswa {
  String nama;
  void sapa() {
    System.out.println("Halo " + nama);
  }
}

Langkah 3: Kondisi

Cek status kelulusan.

int nilai = 82;
if (nilai >= 75) {
  System.out.println("Lulus");
} else {
  System.out.println("Remedial");
}

GUI Swing project

Komponen wajib Swing: JFrame, JButton, JLabel, JTextField. Mini project: calculator GUI.

JFrame frame = new JFrame("Kalkulator");
JTextField input = new JTextField();
JButton btnTambah = new JButton("+");
JLabel hasil = new JLabel("0");
