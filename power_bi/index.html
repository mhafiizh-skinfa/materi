📉 Power BI

Membangun dashboard interaktif dari data mentah menjadi KPI.

Kelas XI

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Pipeline BI: import data, transform, model, visualisasi.
  • Power Query untuk data cleaning sebelum visualisasi.
  • DAX untuk measure dinamis berbasis filter context.
  • Prinsip dashboard efektif: fokus KPI dan cerita data.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
Power QueryTransformasi data mentah.Perbaiki tipe data dulu.
Data ModelRelasi antar tabel.Gunakan star schema jika bisa.
DAXRumus analitis.Mulai dari measure sederhana.

Alur kerja Power BI

Import data dari Excel/CSV/database, transformasi di Power Query, lalu visualisasi dashboard interaktif.

DAX penting: CALCULATE, SUMX, FILTER.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • Power BI seperti ruang kontrol yang menampilkan indikator penting.
  • DAX seperti rumus pintar yang menyesuaikan filter aktif.

Kasus Dunia Kerja

Business analyst memantau performa cabang toko melalui dashboard Power BI harian.

Visual wajib

Gunakan bar chart, pie chart, line chart, card, dan slicer agar analisis mudah dieksplor pengguna.

🛠️ Section 3: Contoh step-by-step

Langkah 1: Transformasi

Bersihkan data di Power Query.

1. Get Data -> Excel
2. Ubah tipe kolom tanggal
3. Hapus baris kosong

Langkah 2: Model relasi

Hubungkan tabel fakta dan dimensi.

FaktaPenjualan[id_produk] -> DimProduk[id_produk]
FaktaPenjualan[id_tanggal] -> DimTanggal[id_tanggal]

Langkah 3: Measure DAX

Hitung metrik omzet.

Total Omzet = SUM(FaktaPenjualan[total])
RataTransaksi = AVERAGE(FaktaPenjualan[total])

Project sales bulanan

Bangun dashboard penjualan bulanan (sales monthly project) dengan KPI total omzet, tren, dan kategori terlaris.

Total Omzet = SUMX(Sales, Sales[Qty] * Sales[Harga])
Omzet Bulan Aktif = CALCULATE([Total Omzet], FILTER(ALL('Tanggal'), 'Tanggal'[Bulan] = MAX('Tanggal'[Bulan])))
