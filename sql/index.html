📈 SQL (Data Analyst)

Mengubah data mentah menjadi insight lewat query analitis.

Kelas XI

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • SELECT, WHERE, JOIN, GROUP BY, ORDER BY sebagai fondasi analisis.
  • Agregasi metrik: COUNT, SUM, AVG dan DISTINCT.
  • CTE untuk query analitis yang mudah dibaca.
  • Validasi hasil query untuk menghindari insight salah.

SQL untuk analitik

Fokus Aggregate Functions: COUNT, SUM, AVG, MAX, MIN; pengelompokan GROUP BY dan HAVING; relasi tabel via JOIN.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • SQL seperti bertanya tepat ke pustakawan untuk jawaban cepat.
  • JOIN seperti menyatukan dua daftar berdasarkan NIS.

Kasus Dunia Kerja

Data analyst e-commerce membuat laporan omzet per kategori dan top pelanggan.

Query tingkat lanjut

Gunakan subquery, CTE WITH, serta window function ROW_NUMBER, RANK, LAG, LEAD untuk insight tren.

🛠️ Section 3: Contoh step-by-step

Langkah 1: Ambil data

Pilih transaksi 30 hari terakhir.

SELECT id_transaksi, tanggal, total
FROM transaksi
WHERE tanggal >= CURRENT_DATE - INTERVAL 30 DAY;

Langkah 2: JOIN data

Gabungkan pelanggan dan transaksi.

SELECT t.id_transaksi, p.nama, t.total
FROM transaksi t
JOIN pelanggan p ON t.id_pelanggan = p.id_pelanggan;

Langkah 3: Agregasi

Hitung omzet per bulan.

SELECT DATE_FORMAT(tanggal, "%Y-%m") AS bulan, SUM(total) AS omzet
FROM transaksi
GROUP BY bulan
ORDER BY bulan;

Studi kasus penjualan

Analisis sales analysis, monthly trends, dan top 5 products dari data sales.

WITH monthly AS (
  SELECT DATE_TRUNC('month', tanggal) bln, SUM(total) omzet
  FROM sales GROUP BY 1
)
SELECT bln, omzet, LAG(omzet) OVER (ORDER BY bln) prev_omzet
FROM monthly;

SELECT produk, SUM(total) total
FROM sales
GROUP BY produk
ORDER BY total DESC
LIMIT 5;
