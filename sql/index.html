📊 SQL

Kemampuan inti untuk data analyst dalam membaca data database.

Data Analyst

📖 Penjelasan Mendalam

SQL dipakai untuk mengambil, mengolah, dan menganalisis data dari database relasional. Materi mencakup SELECT, WHERE, GROUP BY, JOIN, dan agregasi untuk menjawab pertanyaan bisnis.

💡 Analogi & Contoh

SQL seperti bertanya ke perpustakaan data: kita menanyakan data tertentu, lalu mendapat jawaban tepat.

💻 Contoh Praktik

SELECT kelas, AVG(nilai) AS rata_rata
FROM nilai_siswa
GROUP BY kelas
ORDER BY rata_rata DESC;

✏️ Latihan Soal

  1. Tampilkan 5 siswa dengan nilai tertinggi.
  2. Hitung jumlah siswa per kelas menggunakan GROUP BY.
  3. Gabungkan tabel siswa dan nilai menggunakan JOIN.
