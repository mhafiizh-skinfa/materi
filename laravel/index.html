🧰 Laravel

Framework PHP modern untuk pengembangan fullstack berbasis MVC.

Kelas XI

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Arsitektur MVC: Route-Controller-Model-View.
  • Eloquent ORM untuk query berbasis model.
  • Migration dan seeder untuk versi database.
  • Validasi request dan autentikasi bawaan framework.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
RoutePintu masuk request web/API.Kelompokkan per fitur.
ControllerTempat logika bisnis.Jangan campur tampilan.
ModelRepresentasi tabel database.Atur fillable/guarded.

Laravel inti

Laravel adalah framework PHP berbasis MVC; dipilih karena routing rapi, keamanan baik, dan tooling lengkap.

Instalasi: composer create-project laravel/laravel app atau Laravel Installer.

Struktur folder penting: app/, routes/, resources/views/, database/, public/.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • Laravel seperti rangka rumah siap pakai untuk backend modern.
  • MVC seperti pembagian tugas tim proyek yang jelas.

Kasus Dunia Kerja

Startup edukasi membangun LMS dengan Laravel untuk manajemen kelas, materi, dan pembayaran.

Komponen kerja Laravel

Routing: Route::get(), Route::resource(); Controller CRUD; Model Eloquent relasi hasMany/belongsTo; Blade layout/component/directive; migration & seeder; middleware auth dengan Laravel Breeze.
