Backend & Framework

📖 Penjelasan Mendalam

Laravel menyediakan struktur proyek yang rapi, routing, controller, model, migration, ORM Eloquent, dan sistem autentikasi. Materi ini fokus pada praktik membangun aplikasi CRUD yang maintainable.

💡 Analogi & Contoh

Laravel seperti paket bangunan rumah siap pakai: fondasi, dinding, dan instalasi sudah tersedia, tinggal disesuaikan kebutuhan.

💻 Contoh Praktik

// routes/web.php
Route::get('/halo', function () {
    return 'Halo dari Laravel';
});

✏️ Latihan Soal

  1. Buat project Laravel dan tampilkan halaman welcome custom.
  2. Buat migration tabel `posts` lalu jalankan migrate.
  3. Buat fitur CRUD sederhana untuk data siswa.
