1) Penjelasan Mendalam

Topik Multimedia HTML membahas konsep inti HTML dari dasar sampai implementasi. Fokuskan belajar pada alur: pahami istilah, lihat contoh, lalu latihan ulang dengan kata-kata sendiri.

KomponenPenjelasanContoh di Sekolah
Konsep UtamaBagian teori yang harus dipahami dulu.Guru menjelaskan langkah sebelum praktik di lab.
Aturan PentingHal wajib agar hasil benar dan aman.Selalu cek input sebelum diproses.
Hasil AkhirOutput yang harus bisa diuji.Laporan/fitur berhasil berjalan.

Supaya mendalam, pecah masalah menjadi unit kecil: input → proses → output → evaluasi. Metode ini membantu siswa yang mudah bingung saat debug karena setiap langkah bisa dicek satu per satu.

2) Analogi & Contoh Nyata

Analogi Warung: Multimedia HTML seperti alur pesanan di warung. Pelanggan memberi pesanan (input), penjual memasak (proses), lalu makanan disajikan (output).
Analogi HP: Saat membuka aplikasi di HP, kita menekan tombol (input), sistem menghitung (proses), lalu layar menampilkan hasil (output). Konsep yang sama dipakai di HTML.
Kesalahan umum: Langsung menulis kode tanpa rancangan. Akibatnya siswa bingung saat error muncul karena tidak tahu bagian mana yang salah. Selalu buat langkah kecil dulu.

Contoh nyata Indonesia: sistem absensi sekolah, antrian kantin, dan pendaftaran ekstrakurikuler semuanya butuh alur yang jelas agar data tidak tertukar.

  { nama: "Alya", nilai: 88 },
  { nama: "Bima", nilai: 76 },
  { nama: "Citra", nilai: 92 }
];

const lulus = dataSiswa.filter((item) => item.nilai >= 75);
console.log("Siswa lulus:", lulus);

Contoh 3 (Sulit)

// Contoh 3: Lanjutan HTML
// Komentar: pecah masalah besar jadi fungsi kecil agar mudah debug
function evaluasiBelajar(input) {
  if (!input || input.length === 0) {
    return "Data belum diisi";
  }

  const rataRata = input.reduce((total, item) => total + item.nilai, 0) / input.length;
  return `Rata-rata kelas: ${rataRata.toFixed(2)}`;
}

console.log(evaluasiBelajar(dataSiswa));
