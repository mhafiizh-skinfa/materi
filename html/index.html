🧱 HTML

Menyusun struktur konten web yang semantik, rapi, dan mudah diakses.

Kelas X

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Struktur dokumen HTML: doctype, head, body.
  • Elemen semantic HTML5: <header>, <nav>, <main>, <section>, <article>, <footer>.
  • Komponen form: label, input, button dan atribut penting.
  • Prinsip aksesibilitas seperti alt text dan heading hierarchy.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
SemantikElemen dengan makna struktur.Gunakan sesuai fungsi konten.
FormElemen pengambil input user.Pasangkan label dan id.
Mediaimg/video/audio untuk konten kaya.Isi alt gambar secara deskriptif.

Struktur HTML5 wajib

Dokumen selalu dimulai dari deklarasi <!DOCTYPE html> agar browser membaca standar HTML5.

<!DOCTYPE html>
<html lang="id">
  <head>...</head>
  <body>...</body>
</html>

Gunakan tag semantik: header, nav, main, section, article, footer.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • HTML seperti kerangka rumah sebelum dekorasi.
  • HTML seperti daftar isi buku supaya pembaca paham susunan topik.

Kasus Dunia Kerja

Frontend developer marketplace menyusun struktur halaman produk agar SEO dan aksesibilitas tetap baik.

Atribut penting di proyek nyata

Contoh atribut yang wajib dipahami: id, class, href, src, alt, placeholder, required.
