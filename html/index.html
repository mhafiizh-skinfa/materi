🧱 HTML

Pondasi utama untuk membuat konten web yang rapi dan terstruktur.

Web Development

📖 Penjelasan Mendalam

HTML adalah bahasa markup untuk menyusun struktur konten web. Pada materi ini, siswa mempelajari elemen heading, paragraf, list, gambar, tautan, tabel, form, serta elemen semantik seperti header, main, section, dan footer.

💡 Analogi & Contoh

HTML seperti rangka rumah. Tanpa rangka yang baik, bangunan web tidak kokoh dan sulit dikembangkan.

💻 Contoh Praktik

<!DOCTYPE html>
<html lang="id">
  <head><title>Profil Siswa</title></head>
  <body>
    <header><h1>Profil Saya</h1></header>
    <main><p>Saya siswa SMK Informatika.</p></main>
  </body>
</html>

✏️ Latihan Soal

  1. Buat halaman biodata sederhana dengan elemen semantik.
  2. Tambahkan form pendaftaran dengan input nama dan email.
  3. Buat daftar hobi menggunakan ordered list dan unordered list.
