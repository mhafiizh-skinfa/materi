🐘 PHP

Membangun website dinamis di sisi server dengan logika backend.

Kelas XI

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • PHP berjalan di server untuk menghasilkan halaman dinamis.
  • Dasar sintaks: variabel, kondisi, loop, fungsi.
  • Mengelola request dengan $_GET, $_POST, $_SESSION.
  • Keamanan dasar: validasi input dan prepared statement.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
Routing SederhanaPemetaan URL ke file/controller.Gunakan whitelist halaman.
Form HandlingMenerima data user.Sanitasi semua input.
SessionMenyimpan status login.Atur timeout session.

Dasar PHP wajib

Alur server-side: browser kirim request → web server menjalankan PHP → PHP akses database → response HTML/JSON kembali ke browser.

Setup lokal umum: instal XAMPP, aktifkan Apache & MySQL, simpan proyek di htdocs.

Materi sintaks: variabel, tipe data, operator, kondisional, loop, function, include/require.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • PHP seperti dapur restoran: proses terjadi di belakang layar.
  • PHP seperti petugas administrasi yang memproses formulir.

Kasus Dunia Kerja

Portal akademik memakai PHP untuk login guru, input nilai, dan cetak raport.

Kasus backend web

Proyek login sekolah memakai pemrosesan form, CRUD MySQL (Create/Read/Update/Delete), serta session dan cookie untuk autentikasi.
