🔄 DFD

Memetakan proses, entitas eksternal, data store, dan arus data.

Database

📖 Penjelasan Mendalam

DFD digunakan untuk memodelkan bagaimana data masuk, diproses, disimpan, dan keluar dari sistem. Siswa belajar DFD level 0 hingga level lanjut untuk memahami arsitektur informasi aplikasi.

💡 Analogi & Contoh

DFD seperti jalur distribusi paket: dari pengirim, diproses di gudang, lalu sampai ke penerima.

💻 Contoh Praktik

Contoh DFD Level 0 Sistem Nilai:
Entitas: Guru, Siswa
Proses: Kelola Nilai
Data Store: Data Nilai
Arus Data: Input Nilai, Laporan Nilai

✏️ Latihan Soal

  1. Buat DFD level 0 untuk sistem absensi.
  2. Ubah DFD level 0 menjadi level 1 dengan minimal 3 proses.
  3. Identifikasi data store pada sistem pembayaran SPP.
