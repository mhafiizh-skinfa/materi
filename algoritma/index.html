🧠 Algoritma

Belajar menyusun langkah logis, efisien, dan terukur untuk menyelesaikan masalah.

Kelas X

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Definisi algoritma, ciri algoritma baik (jelas, berhingga, efektif).
  • Pola IPO (Input-Process-Output) untuk memetakan solusi.
  • Pseudocode sebagai jembatan dari ide ke kode.
  • Pengantar kompleksitas waktu untuk memilih solusi efisien.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
InputData awal yang dibutuhkan algoritma.Tentukan format dan batas nilai.
ProcessUrutan langkah pemrosesan.Pecah langkah jadi blok kecil.
OutputHasil akhir yang ditampilkan.Pastikan bisa diverifikasi.

Materi wajib tambahan

  • Ciri algoritma lengkap: memiliki input, menghasilkan output, langkah definiteness (jelas), finiteness (berhingga), dan effectiveness (setiap langkah dapat dikerjakan).
  • Metode penulisan algoritma: bahasa alami (natural language), pseudocode, dan flowchart.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • Seperti resep masakan: urutan salah membuat hasil gagal.
  • Seperti rute ke sekolah: jalur efisien menghemat waktu.

Kasus Dunia Kerja

Tim logistik membuat algoritma rute pengiriman agar biaya operasional turun dan ketepatan waktu naik.

Contoh konteks nyata

Di sekolah, algoritma dipakai untuk menghitung luas persegi panjang, menentukan nilai terbesar 3 angka, dan memvalidasi login user secara konsisten.

🛠️ Section 3: Contoh step-by-step

Langkah 1: Definisi masalah

Tentukan masalah menghitung rata-rata 3 nilai.

1. input n1,n2,n3
2. total <- n1+n2+n3
3. rata <- total/3
4. output rata

Langkah 2: Pseudocode

Rapikan alur agar siap diimplementasi.

Mulai
 input n1,n2,n3
 total <- n1+n2+n3
 rata <- total/3
 output rata
Selesai

Langkah 3: Implementasi JS

Uji hasil dengan data contoh.

// Menghitung rata-rata tiga nilai
const rata = (80 + 75 + 90) / 3;
console.log(rata);

Contoh wajib

// Luas persegi panjang
input p, l
luas <- p * l
output luas

// Nilai terbesar 3 angka
max <- a
if b > max then max <- b
if c > max then max <- c
output max

// Login user
input username, password
if username==dataUser and password==dataPass then "berhasil" else "gagal"
