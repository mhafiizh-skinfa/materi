🎨 CSS

Mengatur tampilan web agar menarik, konsisten, dan responsif di berbagai layar.

Kelas X

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Dasar selector, specificity, inheritance, dan cascading.
  • Box model (margin, border, padding, content).
  • Layout modern memakai Flexbox/Grid.
  • Responsif dengan media query dan unit relatif.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
SelectorMenarget elemen untuk diberi gaya.Utamakan class selector.
Box ModelRuang tiap elemen.Gunakan box-sizing border-box.
ResponsiveAdaptasi tampilan lintas layar.Uji di mobile terlebih dahulu.

Selector lanjutan: :hover, :focus, :nth-child().

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • CSS seperti seragam yang membuat tampilan lebih rapi.
  • CSS seperti dekorasi kelas tanpa mengubah struktur ruangan.

Kasus Dunia Kerja

Tim UI menerapkan design system berbasis CSS agar seluruh halaman produk konsisten.

Kasus UI modern

Layout dashboard biasanya menggabungkan Flexbox untuk baris/kolom cepat, Grid untuk area kompleks, CSS Variables untuk tema, dan media query untuk responsif.

🛠️ Section 3: Contoh step-by-step

Langkah 1: Gaya global

Set font dan warna dasar.

/* Mengatur gaya dasar */
body { font-family: Poppins, sans-serif; color: #1f2937; }

Langkah 2: Layout flex

Menyusun beberapa card.

.cards { display: flex; gap: 12px; flex-wrap: wrap; }
.card { flex: 1 1 220px; }

Langkah 3: Media query

Sesuaikan teks untuk layar kecil.

h1 { font-size: 2rem; }
@media (max-width: 600px) {
  /* Menyesuaikan ukuran judul */
  h1 { font-size: 1.4rem; }
}

Contoh kode wajib

:root { --primary: #667eea; --gap: 12px; }
.menu a:hover { color: var(--primary); }
input:focus { outline: 2px solid var(--primary); }
.list li:nth-child(2n) { background: var(--bg); }

.wrap { display: flex; gap: var(--gap); }
.grid { display: grid; grid-template-columns: repeat(3, 1fr); }
@media (max-width: 768px) { .grid { grid-template-columns: 1fr; } }
