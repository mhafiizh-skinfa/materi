▲ Next.js

Framework React untuk web cepat, SEO-friendly, dan siap produksi.

Kelas XI & XII

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • File-based routing dan App Router pada Next.js.
  • Perbedaan CSR, SSR, SSG, ISR untuk kebutuhan performa/SEO.
  • Server Components dan data fetching modern.
  • Optimasi gambar, font, dan deployment produksi.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
RoutingHalaman otomatis dari struktur folder.Gunakan nama route jelas.
RenderingStrategi render konten.Pilih mode sesuai data.
OptimizationFitur bawaan performa.Gunakan next/image dan caching.

Materi XI wajib

Dasar Next.js: file-based routing, konsep SSR/SSG/CSR (analogi: SSR seperti pesan antar langsung, SSG seperti brosur cetak, CSR seperti merakit di rumah), penggunaan getServerSideProps/getStaticProps, API Routes, dan styling Tailwind CSS.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • Next.js seperti pusat distribusi konten yang memilih jalur tercepat.
  • Mode rendering seperti pilihan metode pengiriman barang.

Kasus Dunia Kerja

Portal berita menggunakan Next.js agar halaman cepat dan SEO tetap kuat.

Materi XII lanjutan

Bahas App Router (layout/loading/error), Server vs Client Components, Prisma ORM, autentikasi NextAuth.js, deployment ke Vercel, dan studi kasus fullstack tugas sekolah CRUD+Auth.

🚀 Materi Lanjutan (Kelas XII)

  • Pendalaman App Router: nested layout, parallel routes, intercepting routes.
  • Strategi rendering lanjutan: SSR, SSG, ISR, revalidate, cache tags.
  • Autentikasi & otorisasi: middleware, role-based access, NextAuth/JWT.
  • Optimasi performa: image/font optimization, streaming, bundle analysis.
  • Deployment production: observability, edge runtime, dan pipeline CI/CD.
