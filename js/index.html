🛠️ Section 3: Contoh step-by-step

Langkah 1: Logika dasar

Hitung total belanja.

// Menghitung total belanja
const total = 12000 * 3;
console.log(total);

Langkah 2: Percabangan

Tambah aturan diskon.

const belanja = 150000;
if (belanja >= 100000) {
  console.log("Diskon 10%");
} else {
  console.log("Tanpa diskon");
}

Langkah 3: Event DOM

Ubah status saat tombol diklik.

document.getElementById("btn").addEventListener("click", () => {
  // Memperbarui status di halaman
  document.getElementById("status").textContent = "Sudah diklik";
});
