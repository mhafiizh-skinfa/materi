🧮 Data

Memahami siklus data dari pengumpulan hingga pengambilan keputusan.

Kelas XI

📖 Section 1: Penjelasan mendalam

Konsep Inti

  • Jenis data: terstruktur, semi-terstruktur, tidak terstruktur.
  • Kualitas data: akurat, lengkap, konsisten, tepat waktu.
  • Data lifecycle: collect, clean, analyze, communicate.
  • Etika data: privasi, persetujuan, dan keamanan.

Ringkasan Komponen

KomponenPenjelasanTips Pemula
Data MentahData asli dari sumber.Jangan langsung dipakai analisis.
Data BersihData setelah cleaning/validasi.Dokumentasikan langkah cleaning.
InsightKesimpulan untuk aksi bisnis.Hubungkan dengan keputusan nyata.

Konsep data wajib

Data adalah fakta mentah untuk diolah menjadi informasi; inilah jawaban sederhana untuk pertanyaan what is data.

Jenis data: kualitatif vs kuantitatif, serta structured vs unstructured.

🌍 Section 2: Analogi & Kasus Dunia Kerja

Analogi Sehari-hari

  • Data seperti bahan baku makanan: kualitas menentukan hasil.
  • Data lifecycle seperti proses produksi dari gudang ke konsumen.

Kasus Dunia Kerja

Tim marketing mengevaluasi efektivitas kampanye iklan berdasarkan data konversi.

Lifecycle data

Alur: collect → clean → analyze → visualize → act. Tahap cleaning meliputi penanganan missing value, duplicate, dan outlier.

🛠️ Section 3: Contoh step-by-step

Langkah 1: Klasifikasi data

Identifikasi tipe data tiap kolom.

Kolom umur: numerik
Kolom kota: kategorikal
Kolom tanggal: datetime

Langkah 2: Cleaning

Tetapkan aturan pembersihan data.

1. Hapus duplikat id_transaksi
2. Isi null kategori dengan "Tidak diketahui"
3. Samakan format tanggal

Langkah 3: Insight

Tarik kesimpulan dari pola data.

Jika 60% omzet dari 2 produk utama,
prioritaskan stok dan promosi kedua produk tersebut.

Analisis dasar

Statistik deskriptif (mean, median, modus, sebaran), tools (Spreadsheet, SQL, Python, BI), serta etika data (privasi, bias, keamanan) harus selalu dibahas.
